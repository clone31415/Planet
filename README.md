d = r^2 = x^2 + y^2

M = other mass (kg)

m = mass (kg)

normalise (x, y) = (x / r, y / r)

---

ma = GMmr^-2

a = GMr^-2

---

a(x) = GMr^-2 * x / r

a(x) = GMxr^-3

---

N * m^2 / kg^2 * kg / m^2

= N / kg

= kg * m / s^2 / kg

= m / s^2
